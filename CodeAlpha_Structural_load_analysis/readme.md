# Structural Load Analysis Task

## Project Overview

This project is completed as part of the CodeAlpha Civil Engineering Internship. The project focuses on analyzing structural loads and understanding how different forces affect the stability and performance of structures.

The main objective of this task is to study various types of structural loads, perform load analysis, and evaluate the behavior of structural components under different loading conditions.

## About Structural Load Analysis

Structural load analysis is the process of determining the forces and loads acting on a structure to ensure that it can safely withstand different conditions.

It helps engineers design stable, durable, and safe structures by evaluating load distribution, stress, and structural response.

## Types of Structural Loads Studied

### 1. Dead Load

Dead load refers to the permanent loads of a structure, including:

- Self-weight of structural elements
- Walls
- Floors
- Roofing materials

### 2. Live Load

Live loads are temporary loads caused by occupants, furniture, vehicles, and movable objects.

### 3. Wind Load

Wind load represents the force exerted by wind on structural surfaces.

### 4. Earthquake Load

Earthquake loads represent forces generated due to seismic activity affecting structural stability.

### 5. Environmental Loads

These include:

- Snow loads
- Temperature effects
- Rain loads
